# 🩸 Blood Bank Management API

A RESTful API built with Django REST Framework to manage blood donors, blood requests, and filtering by blood group and location.



## 🚀 Features

* Donor Registration
* Blood Request System
* Filter donors by blood group & location
* Clean and scalable API design

---

## 🛠️ Tech Stack

* Python
* Django
* Django REST Framework


## 📡 API Endpoints

### Donors

* GET /donors/ → List all donors
* POST /donors/ → Create donor

### Blood Requests

* GET /requests/ → List all requests
* POST /requests/ → Create request


⚙️ Installation


git clone https://github.com/yourusername/blood-bank-api.git
cd blood-bank-api
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
